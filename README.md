# 智慧养老管理系统 java570

## 项目概述

智慧养老管理系统是一款基于html、css和js技术的管理后台。该系统致力于为养老机构提供高效的管理与便捷的服务。

## 技术栈

- HTML
- CSS
- JavaScript

## 功能模块

- 控制面板
- 紧急预警
- 护工调度
- 近期老人健康数据

## 系统角色

- 管理员：负责系统管理与数据监控
- 护工：接收与响应紧急预警，执行护工调度

## 运行环境

- 支持任何现代浏览器

## 部署步骤

1. 使用DW、HBuilder、VSCode等软件编辑源代码
2. 将编辑后的代码部署到服务器

## 目录结构

```
- index.html          # 首页
- css/                # 存放CSS样式文件
- js/                 # 存放JavaScript脚本文件
```

- 纯静态页面，易于维护与部署
